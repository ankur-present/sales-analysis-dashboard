# Sales Analysis Dashboard

# Project Overview
This project analyzes retail sales data to understand revenue, profit,
category performance, state-wise trends, and payment behavior.

The workflow combines:

- **Python (Pandas)** → data cleaning & feature engineering  
- **Microsoft Excel** → dashboard creation & business visualization  

This reflects a real-world data analyst workflow.

---

## Dataset
The dataset contains:

- Order ID, Amount, Profit, Quantity  
- Category & Sub-Category  
- Payment Mode  
- Customer, State, City  
- Order Date  

---

# Data Cleaning & Feature Engineering (Python)

Using **Pandas**, the following steps were performed:

- Handled date formatting  
- Created new columns:
  - Year  
  - Month  
  - Year-Month  
  - Profit Margin %  
- Exported a **cleaned dataset** for dashboarding
